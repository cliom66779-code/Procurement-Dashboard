# 🌍 全球農產品行情監控分析儀表板 (Phase 1)

這是一個專業的農產品行情監控分析工具，採用 **React + TypeScript + Tailwind CSS** 構建，專為決策者設計。目前支援 **棕櫚油 (Palm Oil)**、**黃豆 (Soybeans)**、**小麥 (Wheat)** 與 **糖 (Sugar)** 四大關鍵品項的行情追蹤。

---

## ✨ 核心功能與最新優化

-   **💹 多品項動態切換**：支援棕櫚油、黃豆、小麥、糖的一鍵切換，單位 (MYR/T, USd/bu, USd/lb) 自動適配。
-   **📈 全局均線渲染 (Global 30MA)**：**[優化]** 系統採用「先計算、後裁切」邏輯，確保 30MA 紅線在切換至 6M、1Y 等短區間時，依然能從第一天起完整呈現（無斷層）。
-   **📊 深度歷史數據 (10Y)**：數據同步腳本已模擬圖表點擊操作，支持從 6 個月到 10 年的全範圍行情追蹤。
-   **🔥 視覺化漲跌看板**：優化「今日漲跌」方框，大字顯示**絕對點位變動**，下方同步呈現**百分比與趨勢圖示 (▲/▼)**。
-   **🛡️ 數據防誤刷機制**：採用標籤唯一匹配模式 (Name 欄)，杜絕品項混淆，並針對棕櫚油進行特定日期標籤修復。

---

## 📉 數據判讀與 KPI 燈號標準

本系統的核心 KPI 基於 **「最新報價」** 與 **「30日移動平均線 (30MA)」** 的偏離程度進行自動判斷。

| 燈號顏色 | 偏離比例 (Diff %) | 市場含義 | 建議動作 |
| :--- | :--- | :--- | :--- |
| 🔴 **紅燈 (Danger)** | **高於 30MA > 5%** | **市場過熱 / 短線超漲** | 警惕追漲風險，此時單價顯著偏離平均水平。 |
| 🟡 **黃燈 (Warning)** | **接近 30MA ± 5%** | **區間震盪 / 趨勢盤整** | 價格貼合均線點位，觀察後續突破方向。 |
| 🟢 **綠燈 (Success)** | **低於 30MA > 5%** | **市場超跌 / 具備支撐** | 價格相較平均水平顯著偏低，具備相對安全邊際。 |

## 🚀 快速啟動

### 1. 安裝環境
確保已安裝 Node.js (推薦 v18+)，進入專案目錄後執行：
```bash
npm install
```

### 2. 同步全球報價 (數據大掃除 & 更新)
執行下方指令，系統會自動巡迴 Trading Economics 抓取 10 年份歷史報價：
```bash
npm run sync
```
*腳本路徑：`scripts/fetchData.js`，結果存儲於 `public/data/*.json`。*

### 3. 開啟分析介面
```bash
npm run dev
```
瀏覽器開啟：`http://localhost:5173`

## 🛠️ 維修者指引 (Developer Notes)

1.  **30MA 計算邏輯**：系統會提取快取中的 10 年全部歷史數據計算移動平均，因此圖表首端不因時間範圍縮短而呈現空缺，提供最完整的連續趨勢。
2.  **數據分流**：各品項皆有專屬 JSON 檔案，抓取過程嚴格比對 (Name 欄)，確保數據互不污染。
